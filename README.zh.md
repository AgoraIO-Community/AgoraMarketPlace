# 1.项目介绍
## 1.1 概述
本仓库是声网云市场插件的示例项目，仓库包含以下插件示例项目

* 火山美颜插件
    * [Android](ByteDance/android/)
    * [iOS](ByteDance/ios/)
* 相芯美颜插件
    * [Android](FaceUnity/android/)
    * [iOS](FaceUnity/ios/)
* 微软语音转写、翻译插件
    * [Android](Microsoft/android/)
    * [iOS](Microsoft/ios/)
* 商汤美颜插件
    * [Android](SenseTime/android/)
    * [iOS](SenseTime/ios/)
* 云知声语音转写插件
    * [Android](Unisound/android/)
    * [iOS](Unisound/ios/)
